# Question 1
```
Using MongoDB, 
a. create a collection called transactions in database usermanaged (drop if it already exists)
and bulk load the data from a json file, transactions.json
b. Upsert the record from the new file called transactions_upsert.json in Mongodb.
```


## Step 1: Open MongoDB Shell

Start MongoDB shell:

```bash
mongosh
```

---

# (a) Create Database and Collection, Drop Existing Collection, Import JSON Data

## Switch to Database

```javascript
use usermanaged
```

---

## Drop Existing Collection (if exists)

```javascript
db.transactions.drop()
```

---

## Create Collection

```javascript
db.createCollection("transactions")
```

---

## Import JSON Data from `transactions.json`

Exit from `mongosh` and run the following command in terminal/command prompt:

```bash
mongoimport --db usermanaged --collection transactions --file 01_transactions.json --jsonArray
```

### Explanation

* `--db usermanaged` → database name
* `--collection transactions` → collection name
* `--file transactions.json` → input JSON file
* `--jsonArray` → file contains JSON array

---

# (b) Upsert Records from `transactions_upsert.json`

Use the following command:

```bash
mongoimport --db usermanaged \
--collection transactions \
--file 01_transactions_upsert.json \
--jsonArray \
--mode upsert \
--upsertFields transaction_id
```

---

## Explanation of Upsert Command

* `--mode upsert`
  → updates existing records if matched, otherwise inserts new records

* `--upsertFields transaction_id`
  → matching field used for update

---

# Example

If existing document:

```json
{
  "transaction_id": 101,
  "amount": 500
}
```

and new file contains:

```json
{
  "transaction_id": 101,
  "amount": 900
}
```

then MongoDB updates amount from `500` → `900`.

If transaction_id does not exist, new document is inserted.

---

# Final Commands Summary

## Import Initial Data

```bash
mongoimport --db usermanaged --collection transactions --file transactions.json --jsonArray
```

## Upsert Data

```bash
mongoimport --db usermanaged --collection transactions --file transactions_upsert.json --jsonArray --mode upsert --upsertFields transaction_id
```
